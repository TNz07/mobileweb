คู่มือแนะนำสถานที่สำคัญในมหาวิทยาลัยขอนแก่น

ยินดีต้อนรับสู่ มหาวิทยาลัยขอนแก่น

มหาวิทยาลัยขอนแก่น หรือ มข. เป็นมหาวิทยาลัยแห่งแรกในภาคตะวันออกเฉียงเหนือ มีพื้นที่กว้างขวางและมีสถานที่สำคัญหลายแห่งที่น่าสนใจและเป็นสัญลักษณ์ของชาว มข.

1. ศาลเจ้าพ่อมอดินแดง

ศูนย์รวมจิตใจและความศรัทธาสูงสุดของนักศึกษาและบุคลากรในมหาวิทยาลัยขอนแก่น ผู้ที่มาสักการะส่วนมากประสบความสำเร็จโดยเฉพาะอย่างยิ่งในการเรียนและการสอบ

2. คอมเพล็กซ์

ศูนย์อาหารสำหรับนักศึกษาและบุคลากรในมหาวิทยาลัยขอนแก่น มีทั้งร้านอาหาร ร้านตัดผม ร้านถ่ายเอกสาร รวมถึงสถาบีนกวดวิชาต่างๆ

3. ศูนย์ประชุมอเนกประสงค์กาญจนาภิเษก

อาคารสำหรับจัดกิจกรรมสำคัญระดับมหาวิทยาลัยและระดับชาติ เช่น พิธีพระราชทานปริญญาบัตร งานประชุมสัมมนาขนาดใหญ่ หรือคอนเสิร์ต

4. สวนเกษตร

พื้นที่ขนาดใหญ่ที่ใช้เป็นแหล่งศึกษา วิจัย และปฏิบัติการจริงทางการเกษตรสำหรับนักศึกษา นอกจากนี้ยังเป็นสถานที่ที่ผู้คนส่วนใหญ่นิยมใช้เป็นสถานที่วิ่งออกกำลังกาย หรือทำกิจกรรมต่างๆ
